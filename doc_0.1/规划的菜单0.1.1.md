## 六、这份菜单的「设计保证」

### ✅ 角色不混

* 建模 / 治理 / 使用 / 管理职责清晰

### ✅ 路径不乱

* 自上而下、自下而上在 **语义建模** 内闭环
* 统一汇合到 **语义版本**

### ✅ 能力可生长

* 指标、主数据、语义探索、Agent 都有自然扩展位

### ✅ 对外可讲

一句话就能讲清楚：

> 数据资源管理平台
> = 数据治理（语义 / 质量 / 安全）
>
> * 数据资产
> * 数据服务
> * 智能分析
