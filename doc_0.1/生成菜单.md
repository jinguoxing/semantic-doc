# 语义建模平台功能菜单

## 一、建模管理（Modeling）

### 1. 业务事实梳理（L0）
- **业务事项管理**
  - 新建业务事项
  - 编辑事项信息
  - 删除事项
  - 事项列表查看

- **现实事实梳理**
  - 主体事实管理
    - 新增主体
    - 编辑主体
    - 标记纳入/不纳入
    - 删除主体
  - 行为事实管理
    - 新增行为
    - 编辑行为
    - 标记纳入/不纳入
    - 删除行为
  - 状态事实管理
    - 新增状态
    - 编辑状态
    - 状态必要性判断
    - 删除状态

- **业务边界确认**
  - 责任域划分
  - 纳入范围设置
  - 不纳入原因说明

- **业务唯一性判断**
  - 唯一标识定义
  - 可复用性评估
  - 稳定性判断

  - 删除对象
  - 对象列表查看
  - 对象详情查看
  - 对象引用关系查看

- **对象类型管理**
  - 主体对象（Entity）
  - 行为对象（Event）
  - 状态对象（Snapshot）

- **对象属性管理**
  - 新增属性
  - 编辑属性
  - 删除属性
  - 设置必填/可选
  - 设置主键
  - 设置引用关系
  - 枚举值管理

- **对象关系管理**
  - 定义对象间关系
  - 关系类型配置
  - 关系可视化查看

- **对象版本管理**
  - 版本创建
  - 版本对比
  - 版本回退
  - 变更历史查看

### 3. 场景语义建模（L2）
- **场景管理**
  - 新建业务场景
  - 编辑场景信息
  - 删除场景
  - 场景列表查看
  - 场景详情查看

  - 场景目标设定
  - 场景生命周期定义
  - 触发事件配置
  - 成功标准设置

- **场景对象组合**
  - 选择场景主体对象
  - 添加关联对象
  - 设置对象场景角色
  - 移除场景对象

- **场景语义关系**
  - 定义对象间依赖关系
  - 配置执行顺序
  - 设置派生规则

- **场景动作定义**
  - 新增语义动作
  - 编辑动作信息
  - 设置动作目标对象
  - 配置动作结果
  - 删除动作

- **场景版本管理**
  - 场景版本控制
  - 版本变更追踪

### 4. 语义对齐与校准
- **候选对象管理**
  - 查看候选对象
  - 候选对象详情
  - 候选对象来源追踪

- **对象合并**
  - 相似对象识别
  - 对象合并操作
  - 合并策略配置
  - 合并预览

- **对象拆分**
  - 一表多义识别
  - 对象拆分操作
  - 拆分规则配置

- **字段裁剪**
  - 系统噪音字段识别
  - 字段裁剪操作
  - 保留字段确认

- **冲突检测**
